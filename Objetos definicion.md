Cuando se habla de objeto se quiere 
denominar a una clase o una instancia.

Clase: porción de código donde que 
define el comportamiento y estado. 
	El comportamiento lo fijan los
	métodos (son funciones clásicas) 
	El estado lo fijan las variables 
	q reciben el nombre de atributos. 

Instancias: copias de determinado estado, 
con determinadas característica
